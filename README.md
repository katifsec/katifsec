## `> selected_projects`

### PTdoc — VAPT Report Workspace

A professional VAPT report-generation and assessment-management platform designed for penetration testers, reviewers, team leads, and security consulting teams.

**Core direction:**

* Structured vulnerability reporting
* Finding and evidence management
* Executive-summary generation
* Maker-checker review workflow
* Retesting lifecycle
* Client-ready report export
* Local AI assistance through Ollama
* Context-aware security writing and review

---

### Statix — Static Binary Analysis Tool

Python-based utility for rapid binary triage and indicator extraction.

```text
Capabilities:
├── MD5 and SHA-256 generation
├── ASCII and UTF-16 string extraction
├── Overall and sliding-window entropy analysis
├── Suspicious import detection
├── Registry-key identification
└── Optional Radare2 function extraction
```

---

### MOF — Meta of File

Command-line forensic-analysis utility for inspecting files and uncovering hidden or embedded information.

```text
Use cases:
├── Metadata extraction
├── File hash calculation
├── Embedded-content discovery
├── Hidden-data identification
├── Image and video inspection
├── Document analysis
└── Archive examination
```

---

### TPS — Turbo Port Scanner

Lightweight Python-based network reconnaissance and service-fingerprinting tool.

```text
Focus:
├── Fast port enumeration
├── Service identification
├── Internal assessment support
├── Lightweight execution
└── Rapid attack-surface discovery
```
